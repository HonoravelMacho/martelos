## 🛠️ Manual de Operação CLI (Flags)

**REGRA DE OURO:** No terminal, envolva textos e chaves em aspas e cerquilhas: `" #texto aqui# "`

| Flag | Função |
| :--- | :--- |
| `-c / -d` | Criptografar / Descriptografar |
| `-s` | Alvo: **Texto** (Use "#...#") |
| `-i / -a` | Alvo: **Imagem** (.bmp) / **Áudio** (.wav) |
| `-v` | Alvo: **Vídeo** (AVI Raw - Chiado Clássico) |
| `-k` | **Chaves**: Qtd e Frases entre "#...#" |
| `-f` | **Fórmula**: pi, catalan, apery, clausen, arctan, log3 |
