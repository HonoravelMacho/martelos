# 🔨 Projeto M.A.R.T.E.L.O.S.
**Motor Algébrico Ruptivo Transformacional Experimental Livre e Open Source**

Utilitário de alta performance em C para criptografia baseada em constantes irracionais (Pi, Catalan, Apery, Clausen, Arctan).

## 🚀 Como instalar
1. Instale a dependência: `sudo apt install libgmp-dev cmake build-essential`
2. Clone o repo: `git clone https://github.com/HonoravelMacho/martelos.git`
3. Compile:
   ```zsh
   mkdir build && cd build
   cmake ..
   make
   sudo make install
