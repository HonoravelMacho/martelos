# 🔨 Projeto M.A.R.T.E.L.O.S.
**Motor Algébrico Ruptivo Transformacional Experimental Livre e Open Source**

O M.A.R.T.E.L.O.S. é um utilitário de alta performance escrito em C para criptografia baseada em esteiras numéricas de constantes irracionais (Pi, Catalan, Apery, Clausen, Arctan). Utiliza a biblioteca GMP para cálculos de precisão arbitrária e oferece um sistema de cifra por quiasmo e alavancagem quântica.

## 🚀 Instalação Rápida (Linux & Termux)
Para instalar ou atualizar o motor em uma única linha, abra seu terminal e cole:

```bash
curl -sSL https://raw.githubusercontent.com/HonoravelMacho/martelos/main/install.sh | bash

🛠️ Funcionalidades
Arsenal BBP: Pi, Constante de Catalan, Constante de Apery (Zeta 3), Integral de Clausen e Arctan.
Cache O(1): Geração de esteiras numéricas com persistência em memória para performance extrema.
Quiasmo e Âncora: Sistema de rounds de inversão e profundidade de salto na constante (Âncora Quântica).
Multiplataforma: Suporte nativo para Linux (Pop!_OS, Debian, Ubuntu) e Android (Termux).

📄 Como usar

Após instalar, basta digitar martelos no terminal e seguir as instruções do menu.
Finalize suas entradas de texto com Ctrl + D.
Os arquivos criptografados serão salvos automaticamente na sua pasta de Downloads.
